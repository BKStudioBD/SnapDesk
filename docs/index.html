SnapDesk

One lightweight, fully native macOS menu-bar app — capture, annotate, OCR, color-pick, clipboard, screen-record and clean, all in one place.

Replaces Lightshot + TextSniper + CleanShot X + a clipboard manager + a color picker + a Mac cleaner.

Apple Silicon · macOS 14+ 100% on-device · no network No Electron · tiny binary Open source
Download for macOS View on GitHub

Install in one command

Paste this into Terminal — it downloads, installs to Applications, clears the Gatekeeper flag, and launches. No drag, no warning.

curl -fsSL https://raw.githubusercontent.com/BKStudioBD/SnapDesk/main/install.sh | bash

Prefer manual? Grab the DMG, drag SnapDesk to Applications, and right-click → Open the first time.

Everything it does

📸 Capture & Annotate ⌃1

Drag a region or click a window to snap it, then mark up with arrows, boxes, blur, steps, text and spotlight. Copy, save or pin on top.

🔤 Grab Text (OCR) ⌃2

Drag over any text on screen — recognized on-device (Apple Vision) and copied instantly. Only the selected area dims.

🎨 Pick a Color ⌃3

Magnified eyedropper → copies HEX, RGB, RGBA, HSL, CSS, SwiftUI or NSColor. Keep-sampling mode + recent palette.

📋 Clipboard History ⌃4

Searchable history of text & images. One click copies and pastes into the app you came from. Passwords never saved.

🎥 Record Screen ⌃5

Region or full screen, with audio, mic, webcam bubble, on-device captions and privacy blur. Trim timeline with mid-video cuts + export.

📜 Scrolling Capture ⌃6

Auto-scroll a long page hands-free — SnapDesk stitches it into one tall image and opens the result.

🧹 Cleaner

One click frees inactive RAM and clears the junk you tick — caches, temp, logs, Trash — plus a deep app uninstaller that finds every leftover.

⚙️ Built for macOS

Menu-bar only, rebindable shortcuts, launch at login, four icon styles, instant UI. Pure Swift on Apple frameworks — no dependencies.

🔒

Private by design

OCR, captions, image processing and cleaning all run on your Mac. SnapDesk has no network code — nothing is ever uploaded. Clipboard history ignores anything marked secret, so passwords from your password manager are never stored.
